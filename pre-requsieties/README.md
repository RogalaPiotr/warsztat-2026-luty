# Wymagania wstępne do warsztatów

## Sprzęt
- Laptop z Windows / macOS / Linux (uprawnienia administracyjne do instalacji narzędzi).

## Instalacje (przygotuj przed warsztatem)
- Azure CLI: https://learn.microsoft.com/en-us/cli/azure/install-azure-cli?view=azure-cli-latest
  - Weryfikacja: `az version`
  - Bicep jest wbudowany w Azure CLI (od wersji 2.20.0+)
  - Weryfikacja Bicep: `az bicep version`
- Visual Studio Code: https://code.visualstudio.com/
  - Rozszerzenie Bicep: https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-bicep
- Git: https://git-scm.com/install/

## Dostęp do Azure
- Subskrypcja Azure (może być darmowa): https://azure.microsoft.com/en-us/pricing/free-services
- Do utworzenia subskrypcji potrzebny jest adres e-mail (np. Outlook.com): https://signup.live.com/?lic=1
- Wymagane będzie podpięcie karty kredytowej lub debetowej w celu aktywacji subskrypcji (brak opłat przy bezpłatnych limitach, ale weryfikacja płatności jest obowiązkowa)
